Topics
- GUI Programming (concluded)
- Recursion
- Linear Search
- Binary Search
- Introduction to data structures and algorithms
- Algorithmic time complexity and Big O notation
- Midterm Exam

Learning Objectives
- Add check boxes to a frame using the JCheckBox class.
- Add lists to a frame using the JList and JScrollPane classes.
- Explain how the process of recursion works.
- Identify problems for which a recursive solution is suitable.
- Write recursive algorithms.
- Search a list for data using the linear and binary search algorithms.
- Analyze an algorithm to determine the asymptotic time complexity.
- Determine the order of growth of a function and express it in Big O notation.
